# Keyboard Shortcuts Guide

## Text Formatting

| Shortcut               | Action        |
| ---------------------- | ------------- |
| `Ctrl/Cmd + B`         | Bold          |
| `Ctrl/Cmd + I`         | Italic        |
| `Ctrl/Cmd + U`         | Underline     |
| `Ctrl/Cmd + Shift + X` | Strikethrough |
| `Ctrl/Cmd + E`         | Inline code   |

## Headings

| Shortcut             | Action    |
| -------------------- | --------- |
| `Ctrl/Cmd + Alt + 1` | Heading 1 |
| `Ctrl/Cmd + Alt + 2` | Heading 2 |
| `Ctrl/Cmd + Alt + 3` | Heading 3 |
| `Ctrl/Cmd + Alt + 4` | Heading 4 |
| `Ctrl/Cmd + Alt + 5` | Heading 5 |
| `Ctrl/Cmd + Alt + 6` | Heading 6 |
| `Ctrl/Cmd + Alt + 0` | Paragraph |

## Lists

| Shortcut               | Action               |
| ---------------------- | -------------------- |
| `Ctrl/Cmd + Shift + 8` | Toggle bullet list   |
| `Ctrl/Cmd + Shift + 9` | Toggle numbered list |
| `Ctrl/Cmd + Shift + 7` | Toggle task list     |

## Text Blocks

| Shortcut               | Action            |
| ---------------------- | ----------------- |
| `Ctrl/Cmd + Shift + B` | Toggle blockquote |
| `Ctrl/Cmd + Alt + C`   | Toggle code block |

## History

| Shortcut               | Action               |
| ---------------------- | -------------------- |
| `Ctrl/Cmd + Z`         | Undo                 |
| `Ctrl/Cmd + Y`         | Redo (Windows/Linux) |
| `Ctrl/Cmd + Shift + Z` | Redo (Mac)           |

## Special Actions

| Shortcut           | Action            |
| ------------------ | ----------------- |
| `Ctrl/Cmd + K`     | Add link          |
| `Ctrl/Cmd + Enter` | Insert hard break |
| `Mod-\\`           | Clear formatting  |

## Navigation

| Shortcut        | Action             |
| --------------- | ------------------ |
| `Tab`           | Indent (in lists)  |
| `Shift + Tab`   | Outdent (in lists) |
| `Enter`         | New paragraph      |
| `Shift + Enter` | Soft break         |

| `Ctrl/Cmd + P` | Toggle preview       |
| `Ctrl/Cmd + /` | Toggle dark mode     |

## Task List Shortcuts

When working with task lists:

- `Ctrl/Cmd + Enter` - Toggle task completion
- `Tab` - Indent task item
- `Shift + Tab` - Outdent task item

## Table Shortcuts

When cursor is in a table:

- `Tab` - Move to next cell
- `Shift + Tab` - Move to previous cell
- Use toolbar for advanced table operations

## Quick Insertion

Start a line with these characters for quick formatting:

- `#` followed by space - Heading 1
- `##` followed by space - Heading 2
- `###` followed by space - Heading 3
- `-` or `*` followed by space - Bullet list
- `1.` followed by space - Numbered list
- `[]` followed by space - Task list
- `>` followed by space - Blockquote
- ` ``` ` - Code block

## Pro Tips

1. **Multi-line selection**: Select text across multiple lines and apply formatting
2. **Bubble menu**: Select text to see quick formatting options
3. **Link editing**: Click on a link and use the bubble menu to edit or remove
4. **Image operations**: Click on an image to see resize handles
5. **Table navigation**: Use arrow keys to navigate table cells

## Accessibility

- Use `Tab` to navigate between interactive elements
- `Shift + Tab` to navigate backwards
- `Enter` or `Space` to activate buttons
- Screen readers announce all actions
